# SolidOA API 文档

## 接口说明

### 认证模块 `/api/v1/auth`

| 接口 | 方法 | 说明 | 请求参数 | 响应 |
|------|------|------|----------|------|
| 登录 | POST | 用户登录 | `{username, password}` | `{token, user}` |
| 登出 | POST | 用户登出 | - | `{code: 200}` |
| 刷新Token | POST | 刷新访问令牌 | `{refreshToken}` | `{token}` |

### 系统模块 `/api/v1/system`

#### 用户管理

| 接口 | 方法 | 说明 | 权限 |
|------|------|------|------|
| 获取用户列表 | GET | 分页获取用户 | ADMIN |
| 获取用户详情 | GET `/users/{id}` | 获取单个用户 | 登录用户 |
| 创建用户 | POST | 创建新用户 | ADMIN |
| 更新用户 | PUT `/users/{id}` | 更新用户信息 | ADMIN |
| 删除用户 | DELETE `/users/{id}` | 删除用户 | ADMIN |

## 错误码定义

| 错误码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 参数错误 |
| 401 | 未认证 |
| 403 | 无权限 |
| 404 | 资源不存在 |
| 409 | 冲突（如重复提交） |
| 429 | 请求过于频繁 |
| 500 | 系统异常 |

## API 变更日志

### v1.0.0 (2026-05-24)

**新增接口：**
- `/api/v1/dingtalk/oauth/url` - 获取钉钉 OAuth 跳转 URL

**新增功能：**
- 文件上传支持 Magic Number 校验
- 消息发送支持本地消息表补偿
- 所有接口支持 TraceId 追踪

**安全增强：**
- 文件下载/预览需要权限校验
- Feign 调用传递 TraceId
- JWT Secret 长度校验
